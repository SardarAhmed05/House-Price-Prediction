## ЁЯФз Workflow

### 1. Exploratory Data Analysis
- Inspected feature types, distributions, and missing value patterns
- Identified numeric, nominal, and ordinal features

### 2. Data Preprocessing (Train)
- **Missing value imputation** тАФ median for numeric, mode for categorical, `'None'` for ordinal quality columns
- **Feature engineering:**
  - `HouseAge` тАФ years since built at time of sale
  - `GarageAge` тАФ years since garage was built
  - `WasRemodeled` тАФ binary flag for remodeled homes
  - `HasGarage` тАФ binary flag for garage presence
- **Ordinal encoding** тАФ quality columns (`ExterQual`, `BsmtQual`, `KitchenQual`, etc.) encoded on a `None тЖТ Po тЖТ Fa тЖТ TA тЖТ Gd тЖТ Ex` scale
- **One-hot encoding** тАФ nominal columns (`MSZoning`, `Neighborhood`, `SaleType`, etc.)
- **Label encoding** тАФ binary columns (`CentralAir`, `PavedDrive`)
- **Manual mapping** тАФ `BsmtExposure`, `BsmtFinType1`, `BsmtFinType2`
- **Log transformation** тАФ `SalePrice` transformed using `log1p` to reduce skewness

### 3. Modeling
- Model: **Linear Regression**
- Target: `log1p(SalePrice)`
- Train/test split: 80/20
- Evaluation on validation set:
  - **MSE:** 0.0140
  - **R┬▓ Score:** 0.9143

### 4. Test Preprocessing & Submission
- Applied the same preprocessing pipeline to `test.csv` using encoders fitted on training data only
- Reversed log transformation with `np.expm1()` before submission
- Final predictions saved to `submission.csv`
